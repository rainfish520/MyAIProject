# 特效挂接批量替换工具

## 功能说明

本工具用于批量替换.graph文件中的特效地址，支持以下功能：

1. **自动解析**：自动识别并提取.graph文件中的所有特效地址
2. **批量替换**：支持批量修改特效地址
3. **智能识别**：支持.fx、.particle、.shader等多种特效文件格式
4. **自动备份**：保存时自动创建原文件备份

## 使用方法

### 启动工具

**方法一**：双击 `启动工具.bat` 文件

**方法二**：命令行运行
```bash
python effect_address_replacer.py
```

### 操作步骤

1. **选择文件**
   - 点击"浏览..."按钮选择.graph文件
   - 或直接在输入框中输入文件路径

2. **解析文件**
   - 点击"解析文件"按钮
   - 工具会自动提取所有特效地址并显示在左侧面板

3. **修改地址**
   - 在右侧面板的输入框中修改新的特效地址
   - 默认显示原始地址，可直接编辑

4. **批量替换**
   - 点击"批量替换地址"按钮
   - 工具会将所有修改应用到文件内容

5. **保存文件**
   - 点击"保存文件"覆盖原文件（自动创建备份）
   - 或点击"另存为..."保存为新文件

## 支持的特效文件格式

工具能够识别以下类型的特效文件路径：

- **.fx** - 特效着色器文件
- **.particle** - 粒子系统文件
- **.shader** - 着色器文件
- **.material** - 材质文件
- **.anim** - 动画文件
- **.skeleton** - 骨骼文件
- **.mesh** - 网格文件
- **.texture** - 纹理文件
- **.png/.jpg/.tga** - 图片文件
- **.wav/.mp3/.ogg** - 音频文件

## 注意事项

1. **备份文件**：保存时会自动创建 `.bak` 备份文件
2. **编码格式**：默认使用UTF-8编码
3. **路径格式**：支持相对路径和绝对路径
4. **批量操作**：可以只修改部分地址，未修改的保持原样

## 系统要求

- Python 3.6+
- tkinter（Python标准库，通常已包含）

## 常见问题

**Q: 无法启动工具？**
A: 确保已安装Python 3.6+，并检查tkinter是否可用

**Q: 解析不到特效地址？**
A: 检查文件格式是否正确，或联系开发者添加新的解析规则

**Q: 保存失败？**
A: 检查文件是否被其他程序占用，或是否有写入权限
