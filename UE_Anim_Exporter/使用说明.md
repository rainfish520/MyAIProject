# UE动画批量导出工具 v1.8.0

将 Unreal Engine 项目中的动画资产（`.uasset`）批量导出为 FBX 格式的桌面工具。

## 环境要求

- Python 3.x
- 第三方依赖：`psutil`

```bash
pip install psutil
```

## 启动方式

```bash
python UE_Anim_Exporter_GUI.py
```

## 使用流程

### 1. 配置路径

启动后在「项目设置」和「扫描设置」区域填写以下路径：

| 配置项 | 说明 | 示例 |
|--------|------|------|
| UE项目 | `.uproject` 文件路径 | `H:/BBC_5.7/BBC.uproject` |
| UE编辑器 | `UnrealEditor.exe` 路径 | `F:/Program Files/Epic Games/UE_5.7/Engine/Binaries/Win64/UnrealEditor.exe` |
| Content目录 | 项目的 Content 文件夹 | `H:/BBC_5.7/Content` |
| 相对路径 | Content 下的动画资源子目录 | `AdvancedLocomotionV4/CharacterAssets/MannequinSkeleton/AnimationExamples` |
| 导出目录 | FBX 文件输出位置 | `H:/BBC_About/FBX_Export/` |

> 所有路径都可以通过「浏览」按钮选择，也可以手动输入。

### 2. 扫描动画资产

点击 **「扫描动画资产」** 按钮，工具会递归扫描指定目录下的所有 `.uasset` 文件，并在动画列表中展示。

- 排除列表中的动画会自动过滤（默认排除部分 ALS 基础动画）
- 可在「排除列表」文本框中自行增删，每行一个动画名

### 3. 选择要导出的动画

在动画列表中选择需要导出的项目：

- **全选**：点击「全选」按钮
- **取消**：点击「取消」按钮清除选择
- **多选**：按住 `Ctrl` 点击单个项目，或按住 `Shift` 选择范围

### 4. 选择导出模式

在「导出设置」中选择导出模式：

- **保持原目录结构**：导出的 FBX 文件按照 UE 项目中的目录层级存放
- **全部放在根目录**：所有 FBX 文件扁平化放到导出目录根目录下

### 5. 执行导出

提供两种导出方式：

#### 方式一：复制导出脚本（推荐）

适用于 UE 编辑器已经打开的情况。

1. 点击 **「复制导出脚本」**，Python 脚本自动复制到剪贴板
2. 切换到 UE 编辑器
3. 菜单栏：`Window → Developer Tools → Output Log`
4. 在 Output Log 底部切换到 **Python** 标签
5. 粘贴脚本并按回车执行

#### 方式二：直接导出

适用于 UE 编辑器未打开的情况。

1. 点击 **「导出选中动画」**
2. 工具会以无头模式（`-nullrhi`）自动启动 UE 编辑器并执行导出脚本
3. UE 启动需要等待 2-5 分钟，请耐心等待

### 6. 整理文件（可选）

如果导出后 FBX 文件分散在多个子目录中，点击 **「整理文件到根目录」**：

- 将所有子目录中的 `.fbx` 文件移动到导出目录根目录
- 自动清理移动后产生的空目录

## UE编辑器状态指示

工具顶部会实时显示 UE 编辑器的运行状态：

- 🟢 绿色：UE 编辑器正在运行，并显示关联的项目名
- ⚪ 灰色：未检测到 UE 编辑器进程

点击「刷新」可手动更新检测状态。

## 配置持久化

工具关闭时会自动保存当前配置，下次启动自动加载，无需重复填写路径。
